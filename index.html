Project Goals

Developed a website for a non-profit organization aimed at raising awareness and funds for environmental conservation. The site features educational content, event management, and donation capabilities.

  • Create a Functional Calculator: Enable basic arithmetic operations like addition, subtraction, multiplication, division, and percentage calculations.
  • Enhance User Experience: Design a clean, intuitive, and visually appealing interface for easy usability.
  • Ensure Real-Time Calculations: Implement instant evaluations using JavaScript for seamless performance.
  • Improve Responsiveness: Make the calculator adaptable to different screen sizes and devices.
  • Enhance Interactivity: Provide a smooth user experience with hover effects, button clicks, and a functional clear/reset option.
  • Optimize Code Efficiency: Ensure the code is well-structured, readable, and maintainable for future improvements.

Technologies

Challenges and Solutions

  • Challenge: Handling Invalid Inputs
    • Issue: Users might enter invalid expressions, causing errors in calculations.
    • Solution: Use try-catch in JavaScript to catch errors and display a user-friendly message instead of crashing.
  • Challenge: Display Overflow
    • Issue: Long numbers or expressions may exceed the display size.
    • Solution: Limit input length and use CSS to adjust text size dynamically for better readability.
  • Challenge: Button Responsiveness
    • Issue:Buttons may not work properly on smaller screens.
    • Solution:Use CSS flexbox/grid and media queries to ensure a responsive layout.
  • Challenge: Consecutive Operators
    • Issue:Users might enter multiple operators in a row, leading to incorrect calculations.
    • Solution:Implement JavaScript logic to prevent consecutive operator inputs.
  • Challenge: Floating Point Precision Issues
    • Issue: JavaScript sometimes returns inaccurate decimal results (e.g., 0.1 + 0.2 ≠ 0.3).
    • Solution:Use toFixed() to round results to a fixed decimal place for better accuracy.

Key Features:

  • Real-Time Calculation: Performs arithmetic operations (addition, subtraction, multiplication, division) instantly.
  • Error Handling: Uses try-catch to prevent crashes and display an error message for invalid inputs.
  • Responsive Design: Adjusts layout using CSS flexbox and media queries, ensuring usability on all screen sizes.
  • Prevents Consecutive Operators: Ensures users cannot enter multiple operators in a row, avoiding miscalculations.
  • Floating Point Precision Fix: Uses toFixed(4) to round decimal results accurately, solving JavaScript's floating-point issues.
  • User-Friendly Interface: Styled with modern colors, shadows, and animations for a visually appealing experience.
  • Clear Functionality: A dedicated "C" button allows users to reset the display easily.
  • Prevents Multiple Decimals: Restricts users from entering more than one decimal point in a number.
  • Keyboard-Free Input: Fully functional using only on-screen buttons, making it touch-friendly for mobile users.
Outcome:

The calculator project successfully delivers a fully functional and responsive arithmetic calculator, capable of performing basic operations like addition, subtraction, multiplication, and division. With built-in error handling, it prevents invalid inputs, consecutive operators, and floating-point precision issues, ensuring accurate calculations. The user-friendly interface, enhanced with a modern design, provides a seamless experience across devices, including desktops and mobile screens. The project’s responsive layout and touch-friendly buttons improve accessibility, making it easy to use without a keyboard. Additionally, the structured HTML, CSS, and JavaScript codebase allows for future scalability, enabling the addition of advanced mathematical functions and further improvements.

Digital Clock

This digital clock project displays the current time, date, and day dynamically using HTML, CSS, and JavaScript. It updates the time every second and adjusts the background image based on the current day. The clock features a stylish UI with animated transitions and hover effects. It is fully responsive, ensuring a smooth display on different screen sizes. JavaScript functions handle time calculations and DOM updates for real-time accuracy.

Guess My Number

The Guess My Number project is a simple number-guessing game where players try to guess a randomly generated number between 1 and 20. The game provides instant feedback on whether the guessed number is too high, too low, or correct. Players start with a score of 20, which decreases with each wrong guess. The game also tracks the highest score achieved. A reset button allows players to restart and try again.

Guess My Number Game Application Links

Code: https://github.com/Anila-Reddy/GuessMyNumber

Live link: https://anila-reddy.github.io/GuessMyNumber/

Project Goals

  • Develop an interactive number-guessing game using HTML, CSS, and JavaScript.
  • Enhance user experience with real-time feedback and score tracking.
  • Implement game logic to handle valid and invalid inputs efficiently.
  • Introduce a high score system to encourage replayability.
  • Ensure a responsive and visually appealing interface for a seamless experience.

Client

Anila Reddy

Category

UI/UX Design

Technologies

HTML, CSS, JavaScript

Challenges and Solutions

  • Challenge: Handling Invalid Inputs
    • Issue: Users may enter non-numeric or out-of-range values.
    • Solution: Implement input validation to accept only numbers between 1 and 20, displaying appropriate error messages.
  • Challenge: Updating UI Dynamically
    • Issue: The game's feedback needs to update in real time.
    • Solution: Use JavaScript DOM manipulation to change messages, scores, and styles based on user input.
  • Challenge: Preventing Unlimited Attempts
    • Issue: Players could keep guessing without consequences.
    • Solution: Introduce a decrementing score system, where each incorrect guess reduces the score.
  • Challenge: Storing and Updating High Scores
    • Issue: The game resets, losing the best score.
    • Solution: Store the high score in a variable and update it when a new record is achieved.
  • Challenge: Ensuring a Responsive Layout
    • Issue: UI elements may not display properly on smaller screens.
    • Solution: Use CSS media queries for better responsiveness across different devices.

Key Features:

  • Random Number Generation: The game generates a random number between 1 and 20 for users to guess.
  • User Input & Validation: Players can enter their guesses, and invalid inputs are handled gracefully.
  • Dynamic Feedback: The game provides instant feedback with messages like "Too high!" or "Too low!" based on the guess.
  • Scoring System: Players start with a score of 20, which decreases with each incorrect guess, encouraging strategic guessing.
  • High Score Tracking: The highest score is saved and displayed, motivating users to beat their previous records.
  • Responsive UI: The design adapts to different screen sizes, ensuring a seamless experience on all devices.
  • Reset Option: A "Again!" button allows users to restart the game with a new random number.
Outcome:

The Guess My Number game successfully enhances user engagement by providing an interactive and fun guessing experience. Players can improve their logical thinking by refining their guesses based on feedback, while the scoring and high-score tracking features add a competitive element. The game's responsive design ensures accessibility across different devices, and the reset functionality allows continuous gameplay. Overall, the project delivers an enjoyable, user-friendly, and visually appealing game that encourages replayability.
